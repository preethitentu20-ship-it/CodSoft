# CODSOFT Python Internship Projects

This repository contains the Python projects completed as part of the CodSoft Python Programming Internship.

## Tasks Included

### Task 1: To-Do List Application
A command-line To-Do List application that helps users manage their daily tasks.

#### Features
- Add tasks
- View tasks
- Delete tasks
- User-friendly menu-driven interface

---

### Task 2: Simple Calculator
A basic calculator that performs arithmetic operations on user-provided numbers.

#### Features
- Addition
- Subtraction
- Multiplication
- Division
- Easy-to-use interface

---

### Task 3: Rock Paper Scissors Game
A Python implementation of the classic Rock-Paper-Scissors game where the user plays against the computer.

#### Features
- Random computer choice generation
- Winner determination logic
- Multiple rounds of gameplay
- Interactive command-line interface

---

## Technologies Used
- Python 3

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd CODSOFT
```

3. Run any task using Python:

### To-Do List
```bash
python "To do list.py"
```

### Simple Calculator
```bash
python "Simple Calculator.py"
```

### Rock Paper Scissors
```bash
python "rock paper scissor.py"
```
